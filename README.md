## What You Can Do

### 📁 Manage Your Files — the Way You Want

Everything you'd expect, and a lot you probably haven't seen in a file manager before.

- **Copy, move, delete, and rename** files and folders, including across SD cards and USB drives
- **Select multiple files at once** and act on all of them — move, share, compress, delete — in one tap
- **Batch rename** a whole folder of files at once: add a prefix, a number counter, or use a pattern to rename dozens of files in seconds
- **Drag and drop** files between the two side-by-side panels (more on that below)
- **Long-press any file** for a quick peek — see a photo thumbnail, a text excerpt, or media info without opening it
- **Recycle bin** catches deleted files so you can restore them any time within 30 days, instead of losing them forever
- **File shredder** permanently wipes files with a 3-pass overwrite — nothing can be recovered afterwards
- **Split large files** into smaller parts for easy transfer, then reassemble them when you're ready
- **Edit a file's date** — change when it was last modified, useful for organising photos or documents
- **Pin any folder** to your home screen as a shortcut for instant access

### 🧭 Navigate Without Getting Lost

- **Two panels side by side** — open two folders at once and drag files between them without navigating back and forth
- **Tap any part of the path bar** to jump straight to that folder (e.g. tap "Downloads" in `Storage / Downloads / Podcasts` to go up instantly)
- **Back and forward buttons** for folder navigation, separate from your phone's system back gesture — just like a desktop file manager
- **Each folder remembers its own sort order** — sort your Music folder by name and your Downloads folder by date, and each stays that way
- **Recently visited folders** appear on the home screen so your most-used locations are always one tap away
- **Bookmarks** let you save any path as a permanent favourite, accessible from the side navigation
- **File tags** — attach coloured labels to files for your own custom grouping and filtering
- Three view densities: **compact, normal, or comfortable** — pick how much information you see per row
- **USB drives and SD cards** are detected the moment you plug them in, no refresh needed

### 🔍 Find Anything, Fast

- **Search inside files** — not just file names, but the actual text content of documents, code files, and more (`.txt`, `.json`, `.xml`, `.kt`, `.java`, `.py`, and others)
- **Regex search** — if you know what a regular expression is, you can use the full power of pattern matching to find files
- **Filter by file type** — show only images, videos, audio, documents, archives, APKs, or code files
- **Filter by size** — find tiny files, large files, or anything in between with a slider
- **Filter by date** — narrow results to files modified or created within a specific date range
- **Search history** — your recent searches appear as suggestions so you don't have to retype them
- **Category tiles** on the home screen — tap Images, Videos, Audio, or Documents to instantly browse every file of that type on your device

No need to install five separate apps. 9G Files handles all of these natively:

| **Videos & audio** | Full built-in player with play, pause, seek, and track info — no hand-off to another app |

### 🔐 Keep Your Files Safe and Private

- **Encrypt any file** with AES-256-GCM encryption — the same standard used by banks. Set a password and the file is unreadable to anyone without it
- **Secure Vault** — a biometric-locked hidden folder. Files moved into the vault are completely invisible to any other app
- **App lock** — require a fingerprint or PIN before 9G Files even opens
- **Password-protected ZIP archives** — compress and lock files in an AES-256 encrypted archive
- **Checksum verifier** — confirm that a downloaded file hasn't been corrupted or tampered with, using MD5, SHA-1, or SHA-256
- **File shredder** — 3-pass overwrite that makes deleted files forensically unrecoverable
- **File permissions viewer** — see exactly what read, write, and execute permissions are set on any file, displayed in the standard `rwxr-xr-x` format

### 🌐 Access Files Anywhere

- **Connect to your home NAS or a Windows share** (SMB/Samba) — browse network drives as if they were folders on your phone
- **FTP and SFTP client** — connect to any remote FTP or SFTP server and transfer files in both directions
- **WebDAV client** — works with servers like Nextcloud
- **Google Drive, OneDrive, Dropbox, and Box** — browse and manage cloud storage directly, without needing separate apps (uses Android's built-in Storage Access Framework — no API keys, no extra logins)
- **FTP server** — turn your phone into a file server. Connect from any PC on the same Wi-Fi network and browse your phone's storage in any FTP client or Windows Explorer. A QR code is generated automatically for the address
- **Wi-Fi Direct** — transfer files directly between two Android devices, no router or internet connection needed. Either device can send at any time
- **Cast to TV** — stream any video or audio file to a Chromecast or compatible smart TV over your home Wi-Fi

### ⚙️ Understand and Organise Your Storage

- **Storage treemap** — a visual map of what's using your space, colour-coded by file type. Tap to drill down
- **Duplicate finder** — scans your storage and groups files that are identical so you can review and delete them
- **Large files finder** — lists everything on your device sorted by size so you can quickly find what's taking up room
- **Empty folder cleaner** — finds and removes leftover empty folders that other apps leave behind
- **App manager** — see every installed app, back up its APK file, jump to its settings, or uninstall it
- **Storage breakdown** — a detailed category-by-category view of your internal and external storage usage
- **Print** — send any image or PDF directly to a wireless printer
- **QR code sharing** — generate a QR code for your FTP server address or any file path

### 🎨 Make It Yours

- **Light, Dark, and AMOLED themes** — true black for OLED screens saves battery
- **Custom accent colour** — choose any colour and it applies across the whole app
- **Material You** dynamic colour support on Android 12 and above

## Requirements

- **Android 8.0 (Oreo) or newer** — that's any Android phone from 2017 onwards
- Works on phones and tablets
- SD card and USB OTG support included — no extra setup needed

When you open the app for the first time, it will ask for storage access permission. On Android 11 and above this opens a system settings page — just tap "Allow" to give 9G Files access to all your files. Without this, the app can only see its own folder.

---

## Privacy

9G Files works entirely on your device. It does not:
- Send your files or file names anywhere
- Connect to any external server (except servers you explicitly configure yourself, such as your own FTP or NAS)
- Collect analytics or usage data
- Display advertisements

### Requirements
- Android Studio Panda (2024.1.x) or newer
- JDK 17
- Target device or emulator: API 26+

### Build Steps
1. `File → Open → 9GFiles/` in Android Studio
2. Let Gradle sync — all dependencies resolve from Maven Central and Google Maven
3. Run on a physical device for full storage access testing (emulators have limited file system access)

### Permissions
- **Android 11+** — `MANAGE_EXTERNAL_STORAGE` via Settings intent
- **Android 9–10** — `READ_EXTERNAL_STORAGE` + `WRITE_EXTERNAL_STORAGE`
- **Android 13+** — additionally `READ_MEDIA_IMAGES`, `READ_MEDIA_VIDEO`, `READ_MEDIA_AUDIO`

### Architecture
`MVVM + Repository + Single-Activity Navigation`

All UI state is exposed as `StateFlow` from ViewModels and collected with `repeatOnLifecycle`. Heavy file operations (copy, move, zip, delete) run in a foreground Service to avoid ANR and survive backgrounding. Settings are stored in DataStore; bookmarks, recents, and tags are stored in Room.
